# 🎬 Movie Success Predictor

## 📌 Project Overview
Predict movie gross revenue using Machine Learning on IMDb dataset.
Includes feature engineering, linear regression, and evaluation.

## 🛠 Tools Used
- Python, Pandas, Scikit-learn
- Matplotlib, Seaborn
- Jupyter Notebooks

## 📈 Key Steps
1. Data Cleaning & Preparation
2. Feature Engineering (director reputation, actors, runtime)
3. Linear Regression Model
4. Model Evaluation & Insights

## 📊 Key Insights
- Director reputation strongly affects revenue
- Number of votes and meta score contribute to success
- Linear Regression provides a baseline; tree-based models can improve accuracy
